# WorkbookQuery.Refresh method (Excel)

Refreshes this query and its underlying connection.


## Syntax

_expression_.**Refresh**

_expression_ A variable that represents a **[WorkbookQuery](Excel.WorkbookQuery.md)** object.


## Return value

**Nothing**
