# Model3DFormat.Parent Property (Word)

Returns the parent object for the specified object. Read-only.


## Syntax

 _expression_.**Parent**

 _expression_ An expression that returns a [Model3DFormat](./Word.Model3DFormat.md) object.


## Return value

Object
