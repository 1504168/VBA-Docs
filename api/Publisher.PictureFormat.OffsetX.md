# PictureFormat.OffsetX property (Publisher)

Returns or sets a **Variant** value indicating the vertical offset of the picture from the specified shape. A positive value offsets the picture below the shape; a negative value offsets it above the shape. Read/write.


## Syntax

_expression_.**OffsetX**

_expression_ A variable that represents a **[PictureFormat](Publisher.PictureFormat.md)** object.


## Return value

Variant


## Remarks

Numeric values are evaluated in [points](../language/glossary/vbe-glossary.md#point); strings can be in any units supported by Microsoft Publisher (for example, "2.5 in").
