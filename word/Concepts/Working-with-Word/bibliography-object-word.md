# Bibliography Object (Word)

Represents the list of sources available attached to the document (in the current list) or the list of sources available in the application (in the master list).


## Remarks

Biblographies are composed of sources. Also see the  **[Source](../../../api/Word.Source.md)** object and the **[Sources](../../../api/Word.Sources.md)** collection. For more information, see[Working with Bibliographies](./working-with-bibliographies.md).
