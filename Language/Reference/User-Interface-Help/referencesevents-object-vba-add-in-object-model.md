# ReferencesEvents object (VBA Add-In Object Model)

Returned by the **[ReferencesEvents](referencesevents-property-vba-add-in-object-model.md)** property.

## Remarks

The **ReferencesEvents** object is the source of events that occur when a reference is added to or removed from a [project](../../Glossary/vbe-glossary.md#project). 

The **[ItemAdded](itemadded-event-vba-add-in-object-model.md)** event is triggered after a reference is added to a project.

The **[ItemRemoved](itemremoved-event-vba-add-in-object-model.md)** event is triggered after a reference is removed from a project.
