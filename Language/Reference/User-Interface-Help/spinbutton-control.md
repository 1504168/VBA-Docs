# SpinButton Control



Increments and decrements numbers.

## Remarks

Clicking a  **[SpinButton](spinbutton-control.md)** changes only the value of the **[SpinButton](spinbutton-control.md)**. You can write code that uses the **[SpinButton](spinbutton-control.md)** to update the displayed value of another control. For example, you can use a **[SpinButton](spinbutton-control.md)** to change the month, the day, or the year shown on a date. You can also use a **[SpinButton](spinbutton-control.md)** to scroll through a range of values or a list of items, or to change the value displayed in a text box.
To display a value updated by a  **[SpinButton](spinbutton-control.md)**, you must assign the value of the **[SpinButton](spinbutton-control.md)** to the displayed portion of a control, such as the **Caption** property of a **[Label](label-control.md)** or the **Text** property of a **[TextBox](textbox-control.md)**. To create a horizontal or vertical **[SpinButton](spinbutton-control.md)**, drag the sizing handles of the **[SpinButton](spinbutton-control.md)** horizontally or vertically on the form.
The default property for a  **[SpinButton](spinbutton-control.md)** is the **Value** property.
The default event for a  **[SpinButton](spinbutton-control.md)** is the Change event.
